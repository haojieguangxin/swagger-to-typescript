# swagger-to-typescript README

## 主要功能

此插件主要是用于swagger转换成typescript，现在还在试用阶段

## 插件要求

VSCode 版本 > 1.46.1

## 插件设置

使用插件之前，需要设置swagger地址的域名，可设置多个

![](https://tva1.sinaimg.cn/large/008eGmZEly1goo7wevv1ij314w0u0dp1.jpg)

This extension contributes the following settings:

* `swaggerToTs.swaggerDomains`: swagger访问地址域名配置
